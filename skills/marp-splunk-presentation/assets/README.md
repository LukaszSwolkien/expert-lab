# Skill Assets

This folder contains reusable assets for the `marp-splunk-presentation` skill.

## Included

- `splunk.css` - Baseline Splunk-branded Marp theme.
- `logo-splunk-acc-rgb-k.png` - Black logo (default for light background decks).
- `logo-splunk-acc-rgb-w.png` - White logo (for dark background decks).

## Logo usage

The baseline CSS references:

- `logo-splunk-acc-rgb-k.png` (black logo, for light background decks)

Optional alternative:

- `logo-splunk-acc-rgb-w.png` (white logo, for dark background decks)

These files are bundled with the skill. If you remove them, comment out the `section::before` logo block in `splunk.css`.
